**디렉토리 구조**

**backend-study/**

**├─ DB/mysql/**

**├─ Server/spring/**

**├─ Algorithm/**

**└─ OS/linux/**

- **DB-mysql** → MySQL 쿼리, 인덱스, 트랜잭션, ERD 설계 등
- **Server-Spring** → Spring Core, Boot, MVC, JPA, 실습 코드
- **Algorithm** → 알고리즘 풀이(BOJ/Programmers), 자료구조 정리

**브랜치 전략**

**1. main 브랜치**

- 스터디 최종 정리본
- PR 승인 후에만 반영
- 안정적인 문서/코드만 유지

**2. summary 브랜치**

- 팀 공용 요약/정리본 저장
- 주제별 핵심 개념 모아놓는 브랜치
- Markdown 위주

**3. study-이름 브랜치**

- 개인 자유 학습 공간
- 실습 코드, 예제, 필기, 기록 등 전부 가능
- 원하는 만큼 push 가능

4.feature 브랜치

- 구조 수정
